# multi-tool
Different tools for different purposes :)


The goal is to have one shared menu where a user can select the tools they want to run and then those tools will open in different tmux sessions (or something) so the user can switch between the different tools in a minimal environment. The end goal is to also have those tools sync their logged data to a central server so the user can track long term data and we can create different ways to visualize that data.


I've uploaded the rough draft of the apps that I have so far (all ping related), but any new ideas are welcome.
